# IShop Management

IShop Management это программа для автоматизации работы интернет-магазина.

## Системные требования

Microsoft Windows 10  
Установленный .NET Framework 4.7.2  
2Gb оперативной памяти

## Установка

Создайте базу данных и таблицы с помощью скрипта "SQL_IShop_db.sql"  
Наполните ее данными с помощью скрипта "SQL_IShop_content.sql"  

Путь к базе данных необходимо прописать в файле "IShop Management.exe.config",  
в разделе "<connectionStrings>".

## Запуск

Запустите файл "IShop Management.exe"  
За именем пользователя и паролем обратитесь в Администратору.  
Тестовые имена пользователя и пароль (из скрипта):  
Продажи: sales:123  
Склад: sklad:123
